# ADAT-RS

Bu dizin, [ADAT-RS](../README.md) uygulamasının kaynak kodunu içerir. Wordlist dosyası projede yoktur; yollar arayüzden ayarlanır (ana README’de **Wordlists** bölümüne bakın).

## Derleme

```bash
cargo build --release
```

## Çalıştırma

```bash
# Windows
.\target\release\adat-rs.exe

# Linux / macOS
./target/release/adat-rs
```

Ana dokümantasyon için üst dizindeki [README.md](../README.md) dosyasına bakın.
